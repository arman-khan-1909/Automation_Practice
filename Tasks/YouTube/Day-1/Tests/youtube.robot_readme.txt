1. Play A Video is a user defined keyword, it is like a function which has multiple statements in it
2. We can call this in our Test Case, rather than writing individuals Keywords in Test Case
3. If keyword contains args we need to mention them in this line
4. We can define a variable and pass in Test Case(like calling func), or directly give values in Test Case

Resource File
    1. When we use a user defined keyword from a seperate file it is callled Resource file
    2. Resources file will not contain Test Cases it contains only keywords
    3. We can put there common operations like Opening browser and max window
    4. ../ means go to project level first
